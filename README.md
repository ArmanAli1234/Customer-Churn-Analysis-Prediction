#  Customer Churn Prediction & Analysis

An end-to-end Data Analytics and Machine Learning project focused on analyzing customer behavior and predicting customer churn using classification models. This project also includes an interactive Streamlit web application for real-time churn prediction.

---

#  Project Overview

Customer churn is one of the most critical business problems in industries like telecom, banking, SaaS, and e-commerce. Retaining existing customers is often more cost-effective than acquiring new ones.

This project aims to:

* Analyze customer churn patterns
* Identify important churn-driving factors
* Build a Machine Learning model to predict churn
* Deploy predictions through an interactive Streamlit application

# 📊 Exploratory Data Analysis (EDA)

Performed detailed EDA to understand:

* Customer demographics
* Churn distribution
* Contract types
* Internet services
* Payment methods
* Customer tenure patterns
* Monthly and total charges impact

### Insights Extracted

* Month-to-month customers showed higher churn rates
* Fiber optic users had relatively higher churn
* Customers with higher monthly charges were more likely to churn
* Long-term contract customers were more stable
